
Notas sobre el diseño de las piezas CAD utilizadas en iROB-EA


1) Para el diseño de estas piezas se ha utilizado el software FreecAD.


Descarga:	http://www.freecadweb.org/
Tutorial:	http://spainlabs.com/wiki/index.php?title=Tutorial_r%C3%A1pido_de_FreeCad
		http://www.iearobotics.com/wiki/index.php?title=Dise%C3%B1o_de_piezas_con_Freecad


2) Para la impresion de las piezas se ha utilizado una impresora 3D xyzprinting, modelo da Vinci 1.0.A


Web:		http://www.xyzprinting.com
Producto:	http://us.xyzprinting.com/us_es/Product/da-Vinci-1.0

Todas las piezas se han impreso utilizando como material de impresión plástico, PLA.
